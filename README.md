# Репозиторий с конспектами по математике, физике и русскому языку.

Содержит теорию по алгебре, математическому анализу, геометрии, физике, теории вероятностей и математической статистике, а также русскому языку.

Вся информация взята из открытых источников в интернете или из личных записей в ходе обучения в ГБОУ «Школа №58» в г. Москве.
